bake (title, c1, c2, opts)

Creates (‘bakes’) a duotone filter as an SVG tag that can be inserted into HTML.

Options

The following options are supported in the opts table:

  • count: The number of steps to interpolate between the two colors.
  • exp: An exponent to use when interpolating, (default 1 = linear).

Parameters:

  • title string The “title” of the filter. This, prefixed with ‘duotone_’ becomes the ID to be used in the stylesheet to apply the filter.
  • c1 string The first color given as a hash ‘#’ followed by 6 hex digits
  • c2 string The second color given as a hash ‘#’ followed by 6 hex digits
  • opts table A table of options.
